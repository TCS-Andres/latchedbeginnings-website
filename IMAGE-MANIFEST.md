# Image Manifest

Every photo slot on the site shows a tasteful branded placeholder until a real
file is dropped at the exact path below. Once the file exists, it is used
automatically (no code change needed). Recommended: optimized JPG/WebP, sized at
least 1600px on the long edge, warm and natural, real moms/babies/families.

> Do NOT AI-generate Dr. Kacie's likeness. Use her real photo (already in place).
> For provider scenes, keep clinician faces non-specific (hands / over-shoulder)
> so no stand-in is mistaken for Dr. Kacie.

placeholder labeled with the post topic.
